# README -- Athena Help Source

This directory contains the source for Athena's on-line help files.
The directory structure is as follows:

README.md                 - This file
arachne.help              - Root source file for docs/arachne.helpdb
workbench.help            - Root source file for docs/workbench.helpdb
arachne/*.help            - Arachne-specific help files
athena/*.help             - Help files relating to all Athena applications
util/*.help               - Utility files used by all help documents
util/images/*.png         - Images for use by any help document.
workbench/*.help          - Workbench-specific help files
workbench/images/*.png    - Workbench-specific images
Makefile                  - Controls the build process
