DESCRIPTION

helptool ?options...? file.help ?file.helpdb?

helptool(1) compiles help(5) input into a *.helpdb help database that can be browsed using helpserver(n) and mybrowser(n). For example, 

    $ helptool input.help output.helpdb

If the output file name is omitted, the output file's name will be the same as the input file's name with the extension switched to ".helpdb".

The command takes the following options:

-libdir dirname
Adds an external Tcl library directory to the Tcl auto_path, so that packages can be loaded from it.
-lib "packageName ?packageVersion?"
Loads an external Tcl package, so that its commands can called to help define help pages. If the packageVersion is included, the package name and version must be enclosed in double quotes.
-version version
Specifies the version number of the software for which the on-line is being written. This version number can be included in the text of a help page using the version macro.
