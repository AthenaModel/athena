SYNOPSIS

package require projectgui 6.3.3
namespace import ::projectgui::*
  • textwin pathName ?options...?
  •     -title title
  •     -text text
  • pathName cget option
  • pathName configure option value...

DESCRIPTION

The textwin(n) widget is a toplevel window that displays arbitrary text in a scrolling rotext(n) widget. Incremental search is supported. The usual pattern is create a textwin(n) to hold a particular string, and forget about it; it will go away when the user decides to close it or exits the application.

COMMANDS

This module defines the following commands.

textwin pathName ?options...?

Creates a new textwin widget named pathName, and returns pathName. The widget supports the following options, in addition to those provided by the underlying frame widget:

-title title
Sets the window title.
-text text
Sets the text to be displayed in the window.

INSTANCE COMMAND

textwin(n) widgets have the following subcommands:

pathName cget option
Returns the value of the specified option.
pathName configure option value...
Sets the value of one or more of the object's options.

ENVIRONMENT

projectgui(n) requires Tcl 8.5 or later.

To use projectgui(n) in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
