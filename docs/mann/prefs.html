SYNOPSIS

package require projectlib 6.3.3
namespace import ::projectlib::*
  • prefs init
  • prefs get parm
  • prefs names ?pattern?
  • prefs manlinks
  • prefs manpage
  • prefs help parm
  • prefs set parm value
  • prefs reset
  • prefs list ?pattern?
  • prefs load

DESCRIPTION

prefs(n) manages the Athena user preferences; see prefs(5). The preferences are saved automatically whenever a parameter's value changes to the file ~/.athena/user.prefs.

COMMANDS

prefs(n) provides the following commands:

prefs init
Initializes the module. This should be called once, at application start-up. It defines the full set of preference parameters.
prefs get parm
prefs names ?pattern?
prefs manlinks
prefs manpage
These commands are simply delegated to the underlying parmset(n).
prefs help parm
Returns the parameter's help string.
prefs set parm value
Sets the parameter's value and saves the preferences to disk.
prefs reset
Resets all preference parameters to their default values, and saves them to disk.
prefs list ?pattern?
Lists all parameters and their values. If pattern is given, it's a wildcard pattern; only parameters whose names match the pattern are listed.
prefs load
Loads the preferences from disk, if the preference file exists. This is usually only done at application start-up.
