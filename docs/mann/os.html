DESCRIPTION

The os(n)is responsible for determining the operating system on which Athena is running, and for handling OS-specific details. In particular, it determines where particular files should be stored on the different platforms.

COMMANDS

This module defines the following commands:

os type
Returns the operating system type, one of the os types.
os types
Returns a list of the operating system types: linux, win32, and macosx.
os prefsdir
Returns the name of the directory in which Athena should store preference files and similar user data. Note that this module simply identifies the name of the directory; it does not create it. Application code should usually use the prefsdir(n) module to retrieve the preferences directory.
os workdir
Returns the name of the directory in which Athena should store its working data, e.g., the RDB, debugging log, and so forth. Note that this module simply identifies the name of the directory; it does not create it. Application code should usually use the workdir(n) module to retrieve the working directory.

ENVIRONMENT
