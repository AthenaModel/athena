SYNOPSIS

package require projectgui 6.3.3
namespace import ::projectgui::*
  • mktoolbutton w icon tooltip ?options...?
  • mkaddbutton w tooltip ?options...?
  • mkdeletebutton w tooltip ?options...?
  • mkeditbutton w tooltip ?options...?

DESCRIPTION

This module defines several convenience commands for defining toolbar buttons.

COMMANDS

This module defines the following command:

mktoolbutton w icon tooltip ?options...?

Creates a new ttk::button widget named w, which will be configured with -style Toolbutton. The button will display the given icon, and will display the tooltip when the mouse pointer hovers over it. Any options and their values are passed along to the underlying ttk::button.

Returns w.

Note: the icon must be the name of a Tk image. In addition, it is assumed that the name of the "disabled" version of the icon is just icon with a "d" suffix, e.g., ::marsgui::icon::plus22 and ::marsgui::icon::plus22d. This naming convention is standard for marsicons(n) icons.

mkaddbutton w tooltip ?options...?
This is a wrapper around mktoolbutton that uses the basic marsicons(n) ::marsgui::icon::plus22 "plus sign" icon.
mkdeletebutton w tooltip ?options...?
This is a wrapper around mktoolbutton that uses the basic marsicons(n) ::marsgui::icon::trash22 "trash can" icon.
mkeditbutton w tooltip ?options...?
This is a wrapper around mktoolbutton that uses the basic marsicons(n) ::marsgui::icon::pencil22 "pencil" icon.

ENVIRONMENT

toolbutton(n) requires Tcl 8.5 or later.

Dave Hanks, Will Duquette
