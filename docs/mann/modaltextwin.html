DESCRIPTION

modaltextwin(n) displays a scrolling read-only text widget in a modal window with an "OK" button. It is used for displaying arbitrarily large possibly formatted text messages to the user, and is usually used in MS Windows for exit messages that would otherwise be written to standard out.

COMMANDS

This module defines the following commands:

modaltextwin popup ?option value...?

Pops up the modal text window given the following options. Execution will continue after the user presses the "OK" button in the window.

-title title
Sets the window's title string.
-message text
Sets the displayed text. The text is displayed in a monospaced font, so simple formatting is possible.

ENVIRONMENT

This package requires Tcl 8.5 or later.
