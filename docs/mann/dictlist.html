DESCRIPTION

dictlist(n) is a tool for building up lists of dictionaries, and for converting them to various forms (i.e., JSON).

COMMANDS

::projectlib::dictlist is an oo::class. It provides the following commands.

dictlist create name spec

Creates a new instance of dictlist(n) with the given command name. The instance is for building a list of dictionaries, each of which matches the spec.

Each entry in the spec is a list of two items, the key name and the default value of the key. If the default value is omitted, the key's default value is the empty string. The following code creates a dictlist whose entries will have three keys: this and theother, which default to the empty string, and that, which defaults to the number "5". 

set dl [dictlist create mylist {
    this
    {that 5}
    theother
}]
dictlist new ?spec?
Like dictlist create, but creates an instance with an arbitrary name, returning the name.

INSTANCE COMMANDS

Instances of dictlist(n) provide the following subcommands.

obj add value....
Creates a new dictionary, matching the values on the command line to the keys one-for-one. If there are fewer values than keys, the remaining keys are assigned the value "". It's an error if there are fewer keys than values. The dictionary is appended to the list.
obj addwith ?key value...?
Creates a new dictionary with empty values, and then applies the given keys and values. The keys may be specified with a leading "-"; it will be stripped automatically. It's an error to pass an invalid key. The resulting dictionary is appended to the list.
obj append dict
Appends the dict to the list. The dict is assume to match the object's spec.
obj clear
Resets the object's content to the empty list.
obj dicts
Returns the list of dictionaries.
obj find key value ?key value...?

Returns a list of the items in the dict list that exactly match the given keys and values.

TBD: One could easily define a family of these: "glob" and "regexp" for glob and regexp matches, and "findone", "globone", and "regexpone" for finding the first item in the list, a la SQLite's "onecolumn" command.

obj format ?options...?
Uses dictab(n) to format the list. The options are as for the dictab(n) subcommand of the same name.
obj keys
Returns the list of keys.
obj set list
Sets the list of dictionaries.
