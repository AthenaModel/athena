DESCRIPTION

clipboardx(n) is a clipboard for internal Athena objects, especially bean(n) objects such as blocks, tactics, and conditions.

In response to a <<Cut>> or <<Copy>> event, a widget will use clipboardx clear to erase the clipboard and then use clipboardx set to place the copied data on it using the "tag" for the widget's preferred data type. If the data can be represented in multiple ways, the widget can place it on the clipboard in each of the ways, using the appropriate type tag for each.

Later, in response to a <<Paste>> another widget can request the copied data using the clipboardx get subcommand, specifying the tag that it understands. If there is data available for that tag, it will be returned.

If the widget understands multiple types, it can use the clipboardx has subcommand to determine which if any of the types is available.

COMMANDS

clipboardx(n) provides the following commands for use by the application.

clipboardx clear
Erases all data from the clipboard.
clipboardx get tag
Retrieves any data on the clipboard with the given data type tag. If none, returns the empty string.
clipboardx has tag ?tag...?
Given one or more data type tag names, returns the name of the first tag for which there is clipboard data, or the empty string if no data is found. A client that supports pasting of multiple data types can use this to retrieve the name of its most preferred type.
clipboardx set tag value
Puts the value on the clipboard with the given data type tag, replacing any previous data for that tag.
clipboardx tags
Returns the names of the tags for which data is on the clipboard.
