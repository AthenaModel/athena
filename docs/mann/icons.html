DESCRIPTION

This module defines a number of standard icon images. Each icon is a standard Tk image; all are defined in the ::projectgui::icon namespace. Note that it's necessary to use the fully qualified name!.

Icons are generally square; each name is suffixed with the size of the icon in pixels, e.g., ::projectgui::icon::pencil22 is a 22x22 pixel icon.

ICONS

The module defines the following icons.

::projectgui::icon::check22
A green "checkmark" icon.
::projectgui::icon::pause22
A "pause" icon, two black vertical bars.
::projectgui::icon::pencil22
A pencil, i.e., "Edit".
::projectgui::icon::play22
A "play" icon, a black triangle pointing right.
::projectgui::icon::plus22
A bold plus sign, i.e., "Add".
::projectgui::icon::rewind22
A "rewind to start" icon, a black triangle pointing left into a solid black line.
::projectgui::icon::x22
A large red "X", i.e., "Cancel" or "Delete".
