DESCRIPTION

This man page describes a component of the athena(n) package that provides access to the scenario's economic information.

The component is accessible as a subcommand of the athena(n) and athenadb(n) objects; Public Subcommands are accessible from athena(n) and (usually) from athenadb(n), and Private Subcommands only from athenadb(n).

COMMANDS

Public Subcommands

The adb econ command takes the following subcommands, where adb is an instance of athena(n).

adb econ disable
Disables the economic model. The scenario must be unlocked and idle.
adb econ enable
Enables the economic model. The scenario must be unlocked and idle.
adb econ getcge
Returns the cellmodel(n) object containing the economic model's CGE equations. Modifying or destroying the object will damage the scenario. TBD: We should probably change how this works.
adb econ getsam ?flag?

By default, returns the cellmodel(n) object containing the economic model's SAM equations. Modifying or destroying the object will damage the scenario. TBD: We should probably change how this works.

If flag is given and true, the command creates a new cellmodel(n) containing the SAM equations, with default initial values, and returns it. It is the caller's responsibility to destroy the new cellmodel(n).

adb econ gethist
Returns economic history parameters. TBD: ???.
adb econ report ht
Given an instance of htools(n), formats an HTML report of the current state of the economic model.
adb econ state
Returns the current state, ENABLED or DISABLED.
