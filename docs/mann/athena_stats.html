SYNOPSIS

namespace import ::athena::*
  • adb stats groupsbybsys
  • adb stats moodbybsys t
  • adb stats moodbygroups grps t
  • adb stats satbybsys t clist
  • adb stats satbygroups grps concerns t
  • adb stats satbynb t clist
  • adb stats pbmood t ?local?
  • adb stats pbsat t clist ?local?

DESCRIPTION

This man page describes a component of the athena(n) package that is responsible for computing statistics on request about the current run of Athena. These are values computed from the history data "on the fly" rather than as the program runs.

The component is accessible as a subcommand of the athena(n) and athenadb(n) objects; Public Subcommands are accessible from athena(n) and (usually) from athenadb(n), and Private Subcommands only from athenadb(n).

COMMANDS

Public Subcommands

The adb stats command takes the following subcommands, where adb is an instance of athena(n).

adb stats groupsbybsys
Returns a dictionary of groups by belief system id.
adb stats moodbybsys t
Computes a roll-up of mood by group and belief system and returns a dictionary of composite mood by belief system ID at time t.
adb stats moodbygroups grps t
Computes and returns the mood for the set of civilian grps at time t.
adb stats satbybsys t clist
Computes a roll-up of satisfaction at time t by belief system, where clist is a list of concerns. If clist includes all four concerns, then the result is the same as for moodbybsys. Returns a dictionary of composite satisfaction by belief system ID.
adb stats satbygroups grps concerns t
Computes a roll-up of satisfaction at time t for the specific set of civilian grps and concerns, return the computed value.
adb stats satbynb t clist
Computes a roll-up of satisfaction at time t neighborhood, where clist is a list of concerns. If clist includes all four concerns, then the result is the same as for the nbmood variable, which is always computed automatically. Returns a dictionary of composite satisfaction by neighborhood.
adb stats pbmood t ?local?
Computes the playbox mood across all groups at time t. If the local flag is true (the default) only groups in "local" neighborhoods are included; otherwise, all groups are included. Returns the computed mood.
adb stats pbsat t clist ?local?
Computes a roll-up of satisfaction at time t for the entire playbox, where clist is a list of concerns. If clist includes all four concerns, then the result is the same as for pbmood. Returns the composite satisfaction.

Private Subcommands

None defined.
