DESCRIPTION

helpdomain(n) is a smartdomain(n) domain handler for ahttpd(n) that is used to handle help(5) databases.

COMMANDS

::projectlib::helpdomain is an oo::class. It provides the following commands.

::projectlib::helpdomain create name domain dbtitle helpdb
Creates a new instance of helpdomain(n) with the given command name for the given domain. The domain must begin with "/". The instance will serve help pages and images from the help(5) database whose file name is helpdb. The display title of this particular help database is dbtitle
::projectlib::helpdomain new domain dbtitle helpdb
Creates a new instance of helpdomain(n) with an arbitrarily chosen name. The arguments are as for helpdomain create.

INSTANCE COMMANDS

Instances of helpdomain(n) provide the same subcommands as smartdomain(n). Only the following are useful to clients:

obj ahttpd
Installs the helpdomain for use with ahttpd(n), which should already be initialized.
obj mydomain name
Installs the helpdomain for use with the named mydomain(n). TBD: Not yet implemented.
