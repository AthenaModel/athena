appdir(n) -- Application Directory Utility, submodule of projectlib(n)

SYNOPSIS

package require projectlib 6.3.3
namespace import ::projectlib::*
  • appdir init
  • appdir join ?arg arg...?

DESCRIPTION

appdir(n) is responsible for locating the directory from which Athena is running, and making it available to the application. At start-up, Athena calls appdir init, which locates and returns the directory; thereafter, appdir join can be used to create absolute paths to files in the application directory tree.

COMMANDS

This module defines the following commands:

appdir init
Initializes appdir(n). In particular, it determines, saves, and returns the full path to the application directory.
appdir join ?arg arg...?

Called with no arguments, returns the name of the application directory. Otherwise, returns the path name resulting from joining the arguments to the application directory. For example, the following command returns the absolute path of the on-line help file: 

set helpfile [appdir join docs help athena.helpdb]

ENVIRONMENT

This package requires Tcl 8.5 or later.
