prefsdir(n) -- Preferences Directory Utility, submodule of projectlib(n)

SYNOPSIS

package require projectlib 6.3.3
namespace import ::projectlib::*
  • prefsdir init
  • prefsdir initialized
  • prefsdir join ?arg arg...?

DESCRIPTION

prefsdir(n) is responsible for creating and locating the Athena preferences directory, where user preferences and similar user data are stored. At start-up, Athena calls prefsdir init, which creates the os prefsdir directory. The prefsdir join command can then be used to produce file names within that directry.

COMMANDS

This module defines the following commands:

prefsdir init
Initializes prefsdir(n). In particular, it determines and saves the full path to the preferences directory, and returns the full path to it; in addition, it creates the directory itself.
prefsdir initialized
Returns 1 if the preferences directory has been successfully initialized, and false otherwise.
prefsdir join ?arg arg...?

Called with no arguments, returns the name of the preferences directory. Otherwise, returns the path name resulting from joining the arguments to the preferences directory. For example, the following command returns the absolute path of a file called "prefs.dat" in the preferences directory. 

set prefsFil [prefsdir join prefs.dat]

Note that this command always succeeds, even if the preferences directory could not be successfully created.

ENVIRONMENT

This package requires Tcl 8.5 or later.
