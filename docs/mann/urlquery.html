SYNOPSIS

DESCRIPTION

This module defines the urlquery command for manipulating URL query parameter strings.

COMMANDS

urlquery subcommand ?args...?

All urlquery(n) commands are subcommands of urlquery.

urlquery fromdict dict

Converts a parameter dictionary into a form usables as a URL query string. E.g., it converts the dictionary 

    first John last Doe

into the string 

    first=John+last=Doe

If a parameter's value is the empty string, the "=" portion is left out for that parameter.

Parameters and their values may not contain white space, "+", or "=".

urlquery todict query

Converts a URL query string into a parameter dictionary. E.g., it converts the string 

    first=John+last=Doe

into the dictionary 

    first John last Doe

If a parameter has no "=" value, it goes into the dictionary with an empty value. If a parameter has more than one "=", e.g., 

    first=John=Doe

then the second and any subsequent "=" values are ignored.

urlquery get query parmlist

Converts a URL query string into a parameter dictionary with the full canonical list of parameters, i.e., all parameters in parmlist are included and parameters in the query but not in parmlist are omitted.

The parmlist can be a simple list of parameter names; but as with the proc(n) command each entry can be a pair, {name default}. If parameter name is missing from the query, it will be assigned the given default.

ENVIRONMENT

Requires Tcl 8.5 or later.

To use this package in a Tcl script, the environment variable TCLLIBPATH must include the parent of the package directory.
