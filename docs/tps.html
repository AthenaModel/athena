Third-party Software List

Athena depends on the following third-party software during operations:

  • Windows 7, 32-bit or 64-bit; the application is built for 32-bit Windows.
  • Red Hat Enterprise Linux, Version 5 (though other versions of Linux will probably work just as well).

The following third-party software is required during development, but not during operations:

  • ActiveTcl 8.6.4 (Community Edition), from ActiveState.com.
  • Tcl Dev Kit 5.1 (or later).
  • MinGW, to build binary Tcl extensions on Windows 7.
  • InnoSetup 5, to build the Windows installer.
